Copy Sample.home to %programfiles%\Windows Server\Bin\Addins\Home

How to check that the addin is working:

1) Open the Dashboard and select the Home tab

2) On the Home tab, there should be a list of hyperlinks below the title "Community links." At the bottom of the list will be a link titled "Calc" and and link titled "My website."

3) Clicking on the link titled "calc" will start up the caculator.

4) Clicking on the link titled "My website" will start up the browser and naviage to the Microsoft website.
