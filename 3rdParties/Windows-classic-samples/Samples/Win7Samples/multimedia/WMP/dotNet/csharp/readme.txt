wMPCSharp sample

This sample might require certain digital media files to be added to your library. To do this, run the file named loader.hta, which is located in the Windows Media Player SDK sample media folder, and follow the directions.

This sample demonstrates using the Windows Media Player control in a C# solution. It lets the user pick audio or video digital media items from the library. It shows how to manage transport controls, a position trackbar, handle events, and show status information.
