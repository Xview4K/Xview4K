This sample shows the minimum HTML file needed to embed a Windows Media Player ActiveX control (ocx). You can use this file as a template.

This sample might require certain digital media files to be added to your library. To do this, run the file named loader.hta, which is located in the Windows Media Player SDK sample media folder, and follow the directions.

The two buttons play and stop a known digital media file. 

See the Windows Media Player SDK for more information about the Windows Media Player ActiveX control.
