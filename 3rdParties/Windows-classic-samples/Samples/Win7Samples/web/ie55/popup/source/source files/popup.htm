Creating Dynamic Pop-up Elements

The new pop-up feature in Microsoft® Internet Explorer 5.5 provides an easy way to create context menus, dialog boxes, pop-up tips, and glossary entries. These all can be built with DHTML.

Right-click anywhere within the main document to display a customized context menu that uses a pop-up window. Choosing an option in the context menu sends you to the related page in the IFRAME below.




Click on the bold elements in the paragraph below to see glossary entries created using a pop-up window.

Creating Dynamic Pop-up Elements with IE 5.5 is easy now that the developers at Microsoft have implemented it into the latest version of their Web browser. To learn how to use pop-up and other DHTML features, see the MSDN Web Workshop.
