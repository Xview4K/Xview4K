==================================
DHTML Behaviors Library - Menu
==================================
Last Updated: OCT. 18,  2001              


SUMMARY
========
The menu behavior allows authors to create expanding and collapsing menus on 
Web pages. Each menu node can enclose other nodes or standard HTML elements, 
such as an anchor.  


BROWSER/PLATFORM COMPATIBILITY
===============================
Behaviors are supported in Microsoft® Internet Explorer 5 and greater for the Win32® and Unix platforms. 
Any browser that doesn't support behaviors will ignore them.
