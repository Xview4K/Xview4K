rowover Behavior

This document describes technologies available in Internet Explorer 5. While we encourage you to evaluate these features and to send us your feedback, please note that these features are subject to change.

The rowover behavior enables alternate row shading and row highlighting for TABLE objects

The rowover.htc file can be downloaded from the rowover sample page.

The most up to date documentation on the rowover behavior can be found on the World Wide Web. Use Internet Explorer to navigate to the behaviors library reference.

If you have the latest MSDN-Online Web Workshop CD (formerly Sitebuilder Web Snapshot CD), map a drive to the CD and use Microsoft Windows Explorer to navigate to \workshop\author\behaviors\library\rowover\rowover.htm.
