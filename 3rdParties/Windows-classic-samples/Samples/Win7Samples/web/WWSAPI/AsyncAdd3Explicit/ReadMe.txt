AsyncAdd3ExplicitExample

Description
===========

This example illustrates implementing a complex asynchronous function without using WsAsyncExecute.

The example implements an asynchronous function 'AddThree' which adds three integers together by
building upon an existing Add function which can add two integers asynchronously.

Prerequisites
=============

In order to run this sample on Windows XP, Windows Vista, Windows Server 2003
and Windows Server 2008, you may need to install a Windows Update that contains
the runtime DLL for Windows Web Services API. Please consult with the 
documentation on MSDN for more information.

Building the Sample
===================

To build the AsyncAdd3ExplicitExample sample
  1. Open the solution AsyncAdd3ExplicitExample.sln in Visual Studio. 
  2. Change the active solution platform to the desired platform in the 
     Configuration Manager found on the Build menu.
  3. On the Build menu, click Build. 

Running the Sample
==================

To run the AsyncAdd3ExplicitExample sample
  1. Run AsyncAdd3ExplicitExample by clicking Start Without Debugging on the Debug menu.
