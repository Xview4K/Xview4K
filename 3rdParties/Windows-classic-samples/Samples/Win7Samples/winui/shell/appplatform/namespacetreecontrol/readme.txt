NameSpaceTreeControl
=============================================
Demonstrates how to implement a custom namespace tree contol for your application.

This sample also demonstrates how to optionally take advantage of new styles offered by INameSpaceTreeControl2 when running on a version of windows that supports it.

Sample Language Implementations
=============================================
C++

Files:
=============================================
NameSpaceTreeSample.cpp
NameSpaceTreeSample.rc
NamespaceTreeSDKSample.sln
NamespaceTreeSDKSample.vcproj
resource.h

To build the sample using the command prompt:
=============================================
     1. Open the Command Prompt window and navigate to the NamespaceTreeControl directory.
     2. Type msbuild NamespaceTreeSDKSample.sln.


To build the sample using Visual Studio (preferred method):
===========================================================
     1. Open Windows Explorer and navigate to the NamespaceTreeControl directory.
     2. Double-click the icon for the NamespaceTreeSDKSample.sln file to open the file in Visual Studio.
     3. In the Build menu, select Build Solution. The application will be built in the default \Debug or \Release directory.


To run the sample:
=================
     1. Navigate to the directory that contains the new executable using the command prompt.
     2. Run NameSpaceTreeControl.exe
