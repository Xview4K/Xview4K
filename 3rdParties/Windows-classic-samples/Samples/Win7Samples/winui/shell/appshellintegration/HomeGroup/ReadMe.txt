HomeGroup Sample
================
This sample demonstrates how to determine homegroup membership status, enumerate top level items in the HomeGroup Shell Folder, and launch the HomeGroup Sharing Wizard.

Sample Language Implementations:
================================
C++

Files:
======
HomeGroup.sln
HomeGroup.vcproj
HomeGroup.cpp


To build the sample using the command prompt:
=============================================
     1. Open the Command Prompt window and navigate to the HomeGroup directory.
     2. Type msbuild HomeGroup.sln.


To build the sample using Visual Studio (preferred method):
===========================================================
     1. Open Windows Explorer and navigate to the HomeGroup directory.
     2. Double-click the icon for the HomeGroup.sln file to open the file in Visual Studio.
     3. In the Build menu, select Build Solution. The application will be built in the default \Debug or \Release directory.


To run the sample:
=================
     1. Navigate to the directory that contains the new executable, using the command prompt or Windows Explorer.
     2. Type HomeGroup.exe at the command line, or double-click the icon for HomeGroup.exe to launch it from Windows Explorer.
