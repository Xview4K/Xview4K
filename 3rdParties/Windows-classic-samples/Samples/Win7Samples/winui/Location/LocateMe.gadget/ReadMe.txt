=========================
LocateMe Sidebar Gadget
Windows 7 Location Platform
=========================
This sample gadget shows how to use the Location Platform to retrieve the current location and display that on a map.

===============================
Sample Language Implementations
===============================
     This sample is available in the following language implementations:
     JavaScript

============= 
Prerequisites
=============
Windows 7

=======
Running
=======
To run the sample:
     1. Copy the LocateMe.gadget sample folder to %USERPROFILE%\AppData\Local\Microsoft\Windows Sidebar\Gadgets.
     2. Open Control Panel -> Programs -> Gadgets.
     3. Right-click on 'LocateMe' and select 'Add'.
