========================================================================
   SAMPLE : Fax Notify
========================================================================

Description:
-----------------

This sample demonstrates the usage of Fax Server and Account Notifications.

This is supported for Windows Vista.

Usage:
---------

To receive Fax Server Notifications,  run "FaxNotifyCPP.exe /s FaxServer /o 0"
To receive Fax Account Notifications,  run "FaxNotifyCPP.exe /s FaxServer /o 1"

The application then waits for 2 minutes to receive the events. After the 2 minutes the application exists
and no more events are received.
