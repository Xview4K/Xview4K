========================================================================
       Windows Application : Enhanced Storage Sample
========================================================================
Demonstrates working with Enhanced Storage API using portable devices API

Sample Language Implementations:
===============================
     This sample is available in the following language implementations:
     C++

Prerequisites:
===============================
For Windows Vista Enhanced Storage support should be installed first.

To build the sample using the command prompt:
=============================================
     1. Open the Command Prompt window and navigate to the directory.
     2. Type msbuild EhStorManaged.sln.

To build the sample using Visual Studio 2008 (preferred method):
================================================
     1. Open Windows Explorer and navigate to the directory.
     2. Double-click the icon for the .sln (solution) file to open the file in Visual Studio.
     3. In the Build menu, select Build Solution. The application will be built in the default \Debug or \Release directory.

To run the sample:
=================
     1. Navigate to the directory that contains the new executable, using the command prompt or Windows Explorer.
     2. Type EhStorEnumerator2 at the command line, or double-click the icon for EhStorEnumerator2.exe to launch it from Windows Explorer.
