Sample: 
Query WinSAT

Files: 
QueryWinSAT.vcproj
QueryWinSAT.sln
QueryWinSAT.cpp

Demonstrates:
The usage of WinSAT's APIs for querying assessments.

Command Line: 
QueryWinSAT.exe [args] XPATH

[args]
query - queries the most recent entries
queryallxml - queries any previous assessment with specific XPATH
queryxml - queries only recent assessment with specific XPATH
